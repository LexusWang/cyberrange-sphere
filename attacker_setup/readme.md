# Setup the Attacker Machine
We need to provision a Kali machine as an attack workstation so that it can run the Python attack scripts automatically generated by Aurora to execute the planned attacks. We maintain a separate project to accomplish this task, so if you would like more detailed information, please refer to this [project](https://github.com/LexusWang/Aurora-executor).

If you want to deploy the attack machine as quickly as possible, first log in to the Kali machine and then run the following command:
```bash
mkdir ~/Aurora-executor && cd ~/Aurora-executor
curl -fsSL https://raw.githubusercontent.com/LexusWang/Aurora-executor/main/auto_deploy.sh | bash
```
